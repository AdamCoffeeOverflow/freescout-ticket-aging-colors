## Ticket Aging Colors v1.0.8

### Highlights
- Adds left-side color bars on ticket rows based on how long active tickets have been waiting.
- Per-mailbox thresholds and optional breathing effect.

### Installation
1. Download **AdamTicketAgingColors.zip** from the GitHub Release.
2. Upload to **FreeScout → Manage → Modules → Upload Module**.
3. Activate the module.

### Compatibility
- FreeScout: 1.8.0+
- PHP: 8.0+
